**Collapsed Data Center Fabric**

The Collapsed Data Center Fabric with Juniper Apstra is Juniper’s best practice data center network architecture for small networks and  offers comprehensive guidance on deploying a modern collapsed fabric (also known as collapsed spine) with EVPN-VXLAN. As with all  Juniper data center JVDs, this solution is based on best practices as determined by Juniper’s subject matter experts, including Juniper support teams.

The configuration uploaded are validated configuration and the JVD provides more information on the features and details of the deployment.
